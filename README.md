# Sneaker Drop - Frontend

The client-side React application for the Sneaker Drop real-time marketplace.

**Repo:** [https://github.com/RownokNishat/sneak-drop-frontend](https://github.com/RownokNishat/sneak-drop-frontend)

## Setup
1. `npm install`
2. Create a `.env` file:
   ```env
   VITE_API_URL="http://localhost:3001"
   VITE_WS_URL="http://localhost:3001"
   ```
3. `npm run dev`

## Features
- Real-time stock counts via Socket.io
- 60-second reservation countdown
- Activity feed for recent purchasers
- Admin dashboard for drop management
